GitHub Gist Sync (cross-device)

Not connected. Tasks save locally only.
One-time setup (~5 min):
  1. On GitHub: Settings → Developer settings → Fine-grained tokens → Generate new token. Repository access: "Public repositories (read-only)" is fine. Permissions: scroll to Account permissions → Gists → Read and write. Set expiration to whatever you like (1 year is reasonable). Copy the github_pat_... token.
  2. Create a new private gist at gist.github.com. Filename: tracker-data.json. Content: {}. Click "Create secret gist". Copy the gist ID from the URL — it's the long hex string after your username.
  3. Choose a 4+ digit PIN you'll remember. The PIN encrypts your token so it can be saved on this device. You'll enter it once per app cold-start.
🔒 Sync configured. Enter your PIN to unlock and start syncing.

Backup & Sync

For cross-device use (iPad / iPhone): export from desktop, AirDrop / email the JSON file, then import on mobile. iOS Safari doesn't support the File System Access API, so saves on mobile stay in browser storage only.
